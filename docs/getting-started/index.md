# Getting Started

## What is EMBER?

EMBER is the Ecosystem for Multi-Modal Brain-behavior Experimentation and Research.

The EMBER Archive supports free storage and sharing of public neurophysiological and behavioral data, prioritizing data generated as a part of the [Brain Behavior Quantification and Synchronization (BBQS) Program](https://braininitiative.nih.gov/research/systems-neuroscience/brain-behavior-quantification-and-synchronization-program).


## Account Creation

1. Create a GitHub Account
    - If not already a GitHub user, sign up [here](https://github.com/).
2. Register on EMBER
    - Navigate to the EMBER-DANDI [portal](https://dandi.emberarchive.org/) and click the Login button in the top right corner.
3. Account Review and Approval
    - After registering, you will receive an email confirming that your accounr is under review.
    - While we try to review and approve accounts as soon as possible, this is a manual process and some reviews may take an extended time, especially for new GitHub acounts or non .edu email addresses.
4. Access EMBER
    - Upon approal, you will be able to access EMBER by logging in through the Login button in the top right corner.

For support or questions, please reach out to [help@emberarchive.org](mailto:help@emberarchive.org)
